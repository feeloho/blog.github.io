发表于 2018-07-26 | 分类于 mysql

mysql发布最新的8.0版本今天我们在Windows上面安装

安装

下载地址社区版: https://dev.mysql.com/downloads/mysql/

解压zip包到安装目录

比如我的安装目录是：D:\mysql-8.0.11-winx64
